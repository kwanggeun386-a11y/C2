## 🚀 프로젝트 상세

### test 99 - 구구단 우주 탐험 (Multiplication Space Adventure)

초등학생을 위한 재미있는 구구단 게임입니다. 우주 테마와 함께 수학 실력을 키울 수 있어요!

**기능:**
- 🎯 무작위 구구단 도전 모드
- 🔢 2단 ~ 9단 개별 선택 모드
- ⏱️ 30초 타이머
- ⭐ 점수 시스템
- 🐱 우주 고양이 마스코트

**기술 스택:**
- HTML5
- CSS3 (Glassmorphism, 애니메이션)
- Vanilla JavaScript
